# 🏗️ Vigas--md
Herramienta interactiva para el análisis y visualización de estructuras.

## 🚀 Descripción
Este proyecto es una aplicación web diseñada para facilitar el cálculo de parámetros en vigas. Permite a los usuarios ingresar datos técnicos y obtener resultados visuales de manera rápida.

## 🛠️ Tecnologías utilizadas
* **HTML5** para la estructura.
* **Tailwind CSS** para un diseño moderno y responsivo (móvil).
* **JavaScript** para la lógica de cálculo.

## 📱 Cómo usar
1. Abre el enlace de **GitHub Pages** (una vez configurado).
2. Ingresa los valores de longitud y carga.
3. Presiona "Calcular" para ver los resultados.
